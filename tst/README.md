# TGD2 NW.js Application

TypeScript scaffold project using Rspack and NW.js.

## Setup

```bash
npm install
```

## Development

Build and run with NW.js:

```bash
npm start
```

Build only:

```bash
npm run build
```

Watch mode (for development):

```bash
npm run dev
```

## Structure

- `src/` - TypeScript source code
- `public/` - Static files including index.html
- `build/` - Built output (generated)
