    OptionalviewportMatchingScale

    viewportMatchingScale?: number

    If defined, the framebuffer will automatically match the current viewport size multiplied by viewportMatchingScale.

    for instance, if viewportMatchingScale === 0.5 and the viewport id (640, 480), then the framebuffer will have a size of (320, 240).
