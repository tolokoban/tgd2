onZoomRequest

onZoomRequest: ((this, evt) => boolean)

Zooming can be done by the mouse wheel. In this case, we may want to prevent it when the canvas is not in fullscreen, to let the rest of the page to scroll.

This function, if defined, will be called just before the zoom event. If it returns false, the event is not dispatched.
