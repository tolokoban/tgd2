# ToloGameDev Library for WebGL2

Cette bibliotheque exporte des classes qui commencent toutes par le préfixe
**`Tgd`** (pour ToloGameDev).

-   [TgdScene](#/scene) : Fournit le contexte WebGL2 ainsi qu'un gestionnaire de
    resources.
-   [TgdPainter](#/painter) : Un Painter est responsable de peindre le canvas.
-   [TgdAttributes](#/attributes) : Permet de définir la structure des
    attributes d'un vertex ainsi que ses données associées.
