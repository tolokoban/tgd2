# [TgdPainterFramebuffer](docs/classes/TgdPainterFramebuffer.html) - Anti-aliasing

By default, framebuffers do not use anti-aliasing.
We can see this in the next example where we render a cube in a framebuffer in a low resolution canvas.

Here we use a filter to apply anti-aliasing, but that's not the most performant way.

<Demo />

In the [next step](#msaa) we will use MSAA internal capabilities of the framebuffer itself. So we don't need a filter anymore.
