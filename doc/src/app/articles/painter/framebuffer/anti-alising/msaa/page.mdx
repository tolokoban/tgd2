# [TgdPainterFramebuffer](docs/classes/TgdPainterFramebuffer.html) - Anti-aliasing

Activating anti-aliasing on a `TgdPainterFramebuffer` is easy: just set `antiAliasing` attribute to `true`.

Use the cog icon to switch between aliasing and antialiasing.

You may notice few differences from using the filter:

1. Its faster.
2. It only blurs where it make sense.
