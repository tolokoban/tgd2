# Material

- [Depth](#/articles/mesh/material/depth): convert distance to the camera into the color channels
- [Diffuse](#/articles/mesh/material/diffuse): simple phong shading
- [Face orientation](#/articles/mesh/material/face-orientation): blue when the normals face the camera, red otherwise
- [Flat](#/articles/mesh/material/flat): fast applicatino of a single color
- [FlatTexture](#/articles/mesh/material/flat-texture): fast texture application without any shading
- [Ghost](#/articles/mesh/material/ghost): highlight the faces with a normal perpendicular to the camera direction
- [Glass](#/articles/mesh/material/glass): fast approximation of a semi-transparent glass
- [Global](#/articles/mesh/material/global)
- [Hull](#/articles/mesh/material/hull)
- [Mask](#/articles/mesh/material/mask)
- [Mirror](#/articles/mesh/material/mirror)
- [Normal map](#/articles/mesh/material/normal-map)
- [Solid](#/articles/mesh/material/solid): like Flat, but a bit slower because you can change the color
