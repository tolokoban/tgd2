# TgdGeometrySphereIco

<Demo />

An IcoSphere is a subdivided icosahedron (a 20 faces dice).
Every subdivision multiplies the number of faces by 4.

| Subdivisions | Faces |
|--------------|-------|
|       0      |   20  |
|       1      |   80  |
|       2      |  320  |
|       3      | 1280  |
|       3      | 5120  |
|       n      | n*80  |
