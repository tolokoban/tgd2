# Play / Pause

To animate your game, you need to paint many still images. The best is to be
able to paint 60 images per second.

Everytime you want to paint a single frame, you can call `TgdContext.paint()`
method. But if you want to start ananimation that will try to paint 60 frames
per second, you can use `TgdContext.play()`. And `TgdContext.pause()` will stop
the animation.

Tap the screen in the demo below to start/stop the animation.

<Demo />

Beware that `TgdContext.pause()` just pause the render loop, but it does not freeze the time.
That's why when you hit **play** again, the cube reappears where it should be if you didn't hit **pause**.

If you need to _freeze_ the time, then you should consider the class [`TgdTime`](#/articles/time).
