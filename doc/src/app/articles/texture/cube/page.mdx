# Texture Cube

To create a TextureCube, you need siz square images of the same size.

One easy way to get such images is to get an HDRI environment from
[Polyhaven](https://polyhaven.com/hdris) (or any other free website) and to use
Blender to do the convertion. Here is a [Blender file](assets/cubemap.zip) to do
just that.
