## Available uniforms

- `uniTime`: current time in seconds.
- `uniEffectStrength`: float between 0.0 and 1.0, controlled by the slider.
- `uniAspectRatio`: Aspect ratio (width / height).
- `uniAspectRatioInverse`: Inverse aspect ratio (height / width).
