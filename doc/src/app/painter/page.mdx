# Les Painters

Un **Painter** est une class qui permet de dessiner sur le canvas WebGL2. Vous
pouvez créer le votre en héritant de la class abstraite `TgdPainter` ou utiliser
l'un de ceux fournis par la bibliotheque.

-   [TgdPainterBackground](#background): Dessiner un arrière-plan.
-   [TgdPainterClear](#clear): Remplit l'écran d'une couleur unique.
-   [TgdPainterLogic](#logic): Une fonction à appeler pour chaque frame.
